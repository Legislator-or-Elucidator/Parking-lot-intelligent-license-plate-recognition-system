#### 介绍

对停车场智能车牌识别系统的设计以及进行实现

这是一个基于spring boot + maven + opencv 实现的Demo教程项目

贯穿样本处理、模型训练、图像处理、对象检测、对象识别等技术点

java语言的深度学习项目，在整个开源社区来说都相对较少；拥有完整的训练、检测、识别过程的开源项目更是少之又少！！
